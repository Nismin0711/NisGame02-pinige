## ゲームの概要/ルール
授業に出たくないけど出席したことにしたい！
そんな時に一部の大学生が行うイリーガルな技「ピ逃げ」。
あなたは、ピ逃げの達人となり、友人たちの視線をかいくぐり、学生証をカードリーダーにかざすのだ！
